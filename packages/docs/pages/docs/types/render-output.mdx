# RenderOutput

## in typescript

```ts
export type RenderOutput = {
  fileUrl: string;
  cost: number;
  status: ProgressStatus;
  renderId: string;
};
```

### fileUrl

if render is done, here will be the url to the file

### cost

how much the render costed in USD

### status

the status of the render, `rendering`, `done`,`error` or undefined

### progress

render progress, 0-1
