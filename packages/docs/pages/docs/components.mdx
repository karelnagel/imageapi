# @asius/components

Remotion components that are used inside Asius, but that could be used in any other Remotion project. 

## Installation

import { Tab, Tabs } from "nextra-theme-docs";

      </Tab>
      <Tab>
        ```bash copy 
        npm install @asius/components 
        ```
        </Tab>
      <Tab>
        ```bash copy 
        pnpm add @asius/components 
        ```
      </Tab>
    </Tabs>
