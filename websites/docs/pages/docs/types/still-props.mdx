# StillProps

Props that you will need to pass to the sdk or player for images

## in typescript

```ts
export type MediaProps = {
  width: number;
  height: number;
  frame: number;
  comps: ComponentProps[];
  modifications?: Partial<ComponentProps>[];
};
```

### height

video height

### width

video width

### frame

frame number to render or show

### comps: [ComponentProps](/docs/types/component-props)[]

base components that should be copied from the online editor

### modifications: Partial&lt;[ComponentProps](/docs/types/component-props)&gt;[]

modifications that you want to apply to the base components

```ts
const modifications = [
  {
    id: "id_you_want_to_modify", //Text component
    text: "new text",
  },
  {
    id: "id_you_want_to_modify2", // Map component
    location: {
      lat: 20,
      lng: 60,
    },
  },
];
```
