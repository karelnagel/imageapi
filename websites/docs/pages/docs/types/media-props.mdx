# TemplateType

Props that you will need to pass to the sdk or player.

## in typescript

```ts
export type TemplateType = {
  width: number;
  height: number;
  duration: number;
  fps: number;
  comps: ComponentProps[];
};
```

### height

video height

### width

video width

### duration

video duration in seconds

### fps

video fps

### comps: [ComponentProps](/docs/types/component-props)[]

base components that should be copied from the online editor
